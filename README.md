## 🚀 Overview

This project simulates a real-world satellite defense system using AI-powered anomaly detection to protect against cyber attacks.

## 🔐 Features

- **JWT Authentication** - Secure command verification
- **Fernet Encryption** - Military-grade command encryption
- **Isolation Forest AI** - Learns normal behavior, detects attacks
- **Self-Healing Defense** - Auto-blocks malicious IPs
- **Real-time Dashboard** - Live attack visualization

## 🛠️ Tech Stack

- Python 3.8+
- Flask (Backend API)
- Scikit-learn (Isolation Forest)
- Streamlit (Dashboard)
- Cryptography (Fernet/JWT)
